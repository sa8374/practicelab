# Разработчикам PracticeLab

## Как подготовиться к разработке

Подготовьте своё рабочее место к разработке PracticeLab. Вот чеклист готовности (outcome + как проверить):

- Получен ключ к DeepSeek API или другому провайдеру LLM
- Установлен OpenCode
  - [ ] Удалось запустить OpenCode в корне git-репозитория
- OpenCode подключён к LLM провайдеру
  - [ ] Удалось выбрать в чате с нейросетью интересующего провайдера
  - [ ] Нейросеть отвечает без сбоев
- OpenCode подхватил инструкции из репозитория
  - [ ] Нейросеть нашла спецификацию мета-языка
  - [ ] Нейросеть уверенно отвечает на вопросы по разрабываемому продукту

Что понадобится:

- [Сайт OpenCode. Документация](https://opencode.ai/docs/) — узнать, как установить
- [Сайт OpenCode. Документация](https://opencode.ai/docs/ru/) — узнать, как пользоваться
- [Сайт OpenCode. Провайдеры](https://opencode.ai/docs/ru/providers/) — узнать, как подключить LLM-провайдера
