# PracticeLab

Платформа дистанционного обучения по методике Learning by Doing: курсы, уроки, учебные проекты, ревью.

## Разработчикам

Инструкции и справочная информация для разработчиков PracticeLab собраны в отдельном документе [CONTRIBUTING.md](CONTRIBUTING.md).

## Концепты

Ключевые концепты платформы описаны в карточках концептов в каталоге `concepts/`:

- [Работа ученика](concepts/student_work.md) - результат выполнения учеником учебного задания
- [Курс](concepts/course.md) - упорядоченный набор уроков по теме
- [Урок](concepts/lesson.md) - учебная единица внутри курса
- [Шаг](concepts/step.md) - минимальная единица взаимодействия ученика с уроком
- [Платформа PracticeLab](concepts/platform.md) - платформа для онлайн-обучения
- [Статус публикации](concepts/publication_status.md) - состояние контента в жизненном цикле публикации
- [Чеклист самопроверки](concepts/checklist.md) - список критериев для самостоятельной проверки работы
